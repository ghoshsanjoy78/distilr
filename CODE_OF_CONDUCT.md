# Code of Conduct

distilr follows the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its code of conduct. By participating in this project — opening issues, submitting PRs, reviewing code, or otherwise engaging — you agree to abide by it.

The full text is available at:

- HTML: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
- Plain text: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.txt>

## Scope

This applies in all project spaces — the GitHub repository, issue threads, pull requests, discussions, and any other communication channel where someone is participating on behalf of the project.

## Enforcement

If you witness or experience behavior that violates the Contributor Covenant, please report it privately to the project maintainer:

**Sanjoy Ghosh — ghoshsanjoy@gmail.com**

All reports will be reviewed and handled with discretion. The maintainer will respect the privacy and safety of the reporter.

Consequences of violations follow the **Enforcement Guidelines** section of the Contributor Covenant 2.1, which describes a four-step ladder ranging from a private warning to a permanent ban, applied proportionally.
